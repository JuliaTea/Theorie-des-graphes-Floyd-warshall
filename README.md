## Lancement

```bash
python3 floyd_warshall.py
```

Entrer le numéro du graphe à analyser (ex: `1`), puis `q` pour quitter.

## Format des fichiers

Les graphes sont dans des fichiers `<numéro>.txt` :

```
<nombre de sommets>
<nombre d'arcs>
<départ> <arrivée> <valeur>
...
```
